# Fleet Cheese（舰队象棋）

基于中国象棋的变体海战棋类 — 13×14 交叉点制棋盘，七种舰船单位，巡洋舰远程轰炸（延迟结算、双方公开红框锁定），大模型 AI 对战。

## 游戏规则

### 棋盘
- 13×14 个交叉点（线交点），左下角 (0,0)，右上角 (12,13)
- y=6 与 y=7 之间白色海域分界带
- 红方半场 y≤6，蓝方半场 y≥7
- 指挥区（黑色虚线框）：x∈[4,8]，红方 y∈[0,4]，蓝方 y∈[9,13]

### 棋子（每方 19 枚）

| 名称 | 简称 | 数量 | 走法 |
|------|------|------|------|
| 旗舰 | 旗 | 1 | 正交 1 格，限指挥区内。被吃/轰沉即全局败。同列无遮挡=对脸，主动方判负 |
| 护卫舰 | 护 | 2 | 斜行 1 格，指挥区内或紧邻己方可出区。站位可拦截敌方巡洋舰射击线 |
| 巡洋舰 | 巡 | 3 | 移动：正交直线 1~2 格。**攻击**：沿正交线，须恰好越过 1 颗棋子（任意阵营）为炮架，炮架后方空格为有效目标；公开红框锁定，下回合敌方结束时必定结算（潜艇免疫 / 护卫舰拦截） |
| 驱逐舰 | 驱 | 7 | 前进 1 格或左右 1 格，不可后退。到达对方底线升变为巡洋舰 |
| 战机编队 | 机 | 2 | 马走日，敌方别腿。不能攻击潜艇 |
| 潜艇 | 潜 | 2 | 正交 1~2 格可穿行，吃子仅 1 格。免疫远程轰炸与战机的攻击 |
| 预警机队 | 警 | 2 | 斜向 1~3 格可越子，不过河。全灭后己方巡洋舰无法轰炸 |

### 回合与结算
- 一方回合内，可先用任意艘巡洋舰各自开火一次（不结束回合），随后移动一个棋子（结束回合）
- 巡洋舰开火后当回合不可移动，移动后不可开火
- 轰炸在**敌方下回合结束时**必定结算：目标格有舰船→击沉；空格→落空
- 悔棋：每方 4 次配额，AI 对战智能回退 2 半步耗 1 次。生死斗开关开启后全模式禁悔

## 游戏模式

| 模式 | 说明 |
|------|------|
| **AI 对战** | 与大模型对弈，执红/执蓝可选 |
| **自行推演** | 一人操控红蓝双方 |
| **线下对战** | 面对面，蓝方棋子旋转 180° 显示 |
| **对局再现** | 读取 .qp 棋谱回放，自动播放（4s/2x/4x 倍速） |
| **观 AI 战** | AI 自我对弈，自动读取上局棋谱变换策略 |

## API 与 AI

内置 10 家 API 服务商下拉选择：587（免费试用）/ DeepSeek / 豆包 / 谷歌 Gemini / Grok / ChatGPT / LongCat / Kimi / MiMo / 智谱。支持模型自定义填写、连通性快速测试。设置通过 SharedPreferences 持久化。

AI 协议：将完整规则+棋盘+合法走法列表+棋谱+结算记录发送给大模型，模型回复棋谱格式指令。Debug 模式输出推演分析文本。推理模型的 thinking 模式已显式禁用以避免空返回。

每次 AI 调用自动写入 `app/ai_logs/` 文件夹（Prompt + Response 完整记录）。

## 棋谱格式（.qp）
```
# 舰队象棋棋谱
# 时间：2026-07-19 12:00:00
# 先手：红

1.红，驱（0，5）->（0，6）空
# 结算：
2.蓝，巡（6，11）x（6，9）空，驱（6，8）->（6，7）空
# 结算：
3.红，巡（6，2）x（6，7）驱，驱（2，5）->（2，6）空
# 结算：红方轰（6，7）击沉驱
```
